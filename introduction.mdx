## Overview

Flyde empowers you to create, organize, and manage your project's documentation in one centralized space. You can build structured docs sites with MDX, collaborate with your team, and keep everything up-to-date effortlessly. Whether you manage API references, user guides, or onboarding materials, Flyde streamlines the process so you focus on content, not complexity.

With features like version control integration, search, and customizable themes, Flyde scales from small projects to enterprise needs. Start building your docs today and see how it transforms your workflow.

<Callout kind="info">
  Flyde supports MDX, so you write docs with familiar Markdown and React components.
</Callout>

## Quick Navigation Guide

Get started with Flyde in minutes using these steps.

<Steps>
  <Step title="Create Your Space" icon="plus">
    Log in to your Flyde dashboard at `https://dashboard.example.com` and create a new documentation space for your project.
  </Step>
  <Step title="Add Your First Page" icon="file-text">
    Click "New Page" and use the MDX editor to write content. Add frontmatter with `title` and `description`.
  </Step>
  <Step title="Organize with Folders" icon="folder">
    Drag pages into folders to build your site structure. Preview changes live.
  </Step>
  <Step title="Publish" icon="globe">
    Deploy your docs site with one click. Share the live URL with your team.
  </Step>
</Steps>
