---
title: Cannot Save Flow
description: Fix issues with saving flows in Flyde.
---

## Symptoms
- The "Save" button is disabled
- Error: "Failed to save flow"

## Common Causes
- No project selected
- Lost internet connection

## Quick Fix
<Steps>
  <Step title="Check project">
    Make sure you've opened an existing project or workspace before creating a flow.
  </Step>
  <Step title="Restore connection">
    Confirm you are online. Reconnect and try saving again.
  </Step>
</Steps>
